pydct
=====

An implementation of JPEG's Discrete Cosine Transform in Python.

Usage
-----

To compress a bitmap, run,
```shell
$ ./dct.py -encode data/puppy.bmp
```

To decode the result, run,
```shell
$ ./dct.py -decode data/puppy.dct
```
